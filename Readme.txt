Instructions to play MyoRacer!!
To run this project you must need Myo , Myo Connect Application, and node.js installed on your machine. 
Configure the Myo and place the myoracer directory in the webserver directory.
Use Apache Webserver for better results and open the index.html page to run the game
Mount the device on your right arm
Double tap to connect the device
Hold the Fist tight to decelarate
Stretch fingers to accelarate
Wave in for left
Wave out for right
Enjoy the game!!
Ref:
For this project I have used the race template available on the github
